Kryptos
=======
Requires PHP 5.4 or higher and MySQL to run.

## Setup
git clone https://github.com/OpenWireSec/Kryptos.git

Go to core/Init.php

Put in your database username and password
Save and Run: 

	sudo bash setup.sh
	sudo bash startServer.sh

Go to http://localhost:8000

	Username: root
	Password: KryptosCTF

Enjoy!!
